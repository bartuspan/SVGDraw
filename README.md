## Simple SVG Drawing
Best performance in Chrome (for now)

### Implemented:
 - Circle
 - Ellipse
 - Rectangle
 - Square
 - Stroke Width & Color
 - Fill Color
 - Move Objects
 - Rotate
 - Resize
 - Custom Colors
 - Restore Saved Status
 - Text
 - Font Family
 - Font Size

### Todo:

### Instructions
Watch for changes and compile during development:  
```bash
$ npm install
$ grunt watch
```
Compile all js files using RequireJS for production:  
```bash
$ grunt requirejs
```
